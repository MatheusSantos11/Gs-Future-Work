# Gs-Future-Work

**Resumo do projeto:**

Este repositório contém o projeto "Future Work": uma aplicação frontend construída com React (TypeScript), Vite e Tailwind CSS para demonstrar perfis, projetos e funcionalidades de networking (cards de projeto, modais de perfil, recomendações e customização). O código-fonte principal está na pasta `future-work`.

**Instalação (passo-a-passo):**

Pré-requisitos:

- Node.js (recomendado v16+)
- npm (ou `pnpm`/`yarn` se preferir)

1. Abra um terminal e entre na pasta do projeto:

```git bash
cd future-work
```

2. Instale as dependências:

```git bash
npm install
```

3. Rode a aplicação em modo de desenvolvimento:

```git bash
npm run dev
```

4. Para gerar o build de produção:

```git bash
npm run build
```

5. Para pré-visualizar o build localmente:

```git bash
npm run preview
```

6. Para instalar o pacote de icones

```git bash
npm install lucide-react
```
